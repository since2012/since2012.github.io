## windows
* 注册表（浏览器首页）：first home page
* 镜像劫持，替换系统默认记事本
* 修改背景色

步骤：  1、快捷键Win+R打开运行，输入regedit运行注册表编辑程序；  2、依次打开[HKEY_CURRENT_USER\Control Panel\Colors]，双击修改Window字符串值为“204 232 207”（注意空格），此为豆沙绿护眼色，需要修改为别的颜色可自行输入颜色RGB值；  3、关闭注册表，注销或重启系统即可生效。
